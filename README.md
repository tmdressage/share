# アプリケーション名

## 作成した目的
フロントエンド自己学習用。<br>
フロントエンドにreact.js、バックエンドにlaravel(php)を用いて作成しました。<br>
認証機能にはFirebaseを使用しています。<br>

## 環境構築
**1、Gitのクローン**<br>
<br>
(コマンドライン上)<br>
```
$ git clone git@github.com:tmdressage/share.git
```
<br>

**2、Docker環境の構築**<br>
<br>
(コマンドライン上)<br>
```
$ docker-compose up -d --build
```
<br>

**3、Laravelの設定**<br>
<br>
(コマンドライン上)<br>
```
$ docker-compose exec php bash
```
(PHPコンテナ内)<br>
```
$ composer install
```
<br>

**4、.envファイルの作成**<br>
<br>
(コマンドライン上)<br>
```
$ cd frontend/src/
$ touch .env
$ cd ../../
$ cd backend/src/
$ cp .env.example .env
$ cd ../
$ docker-compose exec php bash
```
(PHPコンテナ内)<br>
```
$ php artisan key:generate
```
<br>

**5、テーブルの作成**<br>
<br>
(PHPコンテナ内)<br>
```
$ php artisan migrate --seed
```
<br>
